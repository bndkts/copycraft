# Deutsche Überschriften & CTAs – native Muster statt Übersetzung

Lies diese Datei, wenn du **deutsche** Überschriften, Hero-Zeilen, Subheadlines oder
Button-Texte schreibst. Sie ist die Ausführungsebene fürs Deutsche; das sprachneutrale
Handwerk steht in `SKILL.md`. Kernprinzip: Ein deutscher CTA wird **native gedacht**, nie
Wort für Wort aus dem Englischen übersetzt. „Get started“ wird zu „Jetzt starten“, nicht
zu „Bekomme gestartet“; „Learn more“ wird zu „Mehr erfahren“, nicht zu „Lernen Sie mehr“.

Alle Beispiele sind **selbst konstruiert** (Domäne: Markenschutz / Brand Protection,
B2B-Security) – keine fremde Copy. Vor Veröffentlichung von einer muttersprachlichen
Texterin prüfen lassen.

## 1. du oder Sie – erst entscheiden, dann texten

Jeder CTA und jede Anrede in der Headline hängt an dieser einen Entscheidung. Quelle der
Wahrheit ist `.claude/brand-brief.md`. Steht dort nichts und sagt die Nutzerin nichts,
**einmal nachfragen** – dann konsequent bleiben.

Die Todsünde ist das **Mischen** von du und Sie innerhalb einer Journey: Hero siezt
(„Steigern Sie Ihre Sicherheit“), das Formular duzt („Erfahre, wie du Zeit sparst“). Bei
einer Security-Marke wirkt das unsauber und schadet dem Vertrauen direkt.

Ohne Vorgabe und bei formellem B2B-Publikum: **Sie** als Default. Ein falsches Sie kostet
weniger als ein falsches du (Strzelecki, CleverReach: „Ein 'Sie' tut niemandem weh“;
Appinio 2019: 61 % sehen Siezen als Respekt). du nur, wenn die Marke bewusst
Gründer/SMB-Self-Serve adressiert und es **überall** durchzieht (so wie sevDesk, Trade
Republic, N26). Anrede ≠ Ton: Man kann warm siezen und steif duzen – Nähe entsteht aus den
Formulierungen, nicht aus dem Pronomen.

## 2. Native CTA-Map (englischer Ursprung → idiomatisches Deutsch)

Englische CTAs werden nicht übersetzt, sondern auf die Wendung **abgebildet**, die ein
deutsches Produkt tatsächlich nutzt.

| Englischer Ursprung | Natives Deutsch |
|---|---|
| Get started / now | Jetzt starten · Jetzt loslegen · Los geht's |
| Learn more | Mehr erfahren · Mehr dazu |
| Sign up (free) | Kostenlos registrieren · Jetzt kostenlos anmelden |
| Start free trial / Try for free | Kostenlos testen · 30 Tage kostenlos testen · Kostenlos ausprobieren |
| Book / Request a demo | Demo vereinbaren · Demo anfragen · Demo buchen · Live-Demo ansehen |
| Contact sales | Vertrieb kontaktieren · Sales-Team kontaktieren · Kontakt aufnehmen |
| Buy now | Jetzt kaufen |
| Download | Jetzt herunterladen · Hol dir die App |
| Get a quote | Angebot anfordern |

Security-spezifische CTAs (konstruiert): „Marke jetzt prüfen lassen“ · „Markenschutz
starten“ · „Bedrohungen prüfen lassen“ · „Fake-Shops melden“.

## 3. CTA-Handwerk

- **Aktionsverb zuerst:** Hol dir, Starte, Teste, Sichere dir, Buche, Schütze. Der Button
  beschreibt eine Handlung, kein Substantiv.
- **Nutzen anhängen:** „Jetzt starten & 20 % sparen“ schlägt das bloße „Jetzt starten“.
  Der Button verrät, was der Klick bringt.
- **Kein schwaches Passiv:** „Hier können Informationen erhalten werden“ → „Hol dir jetzt
  alle Infos“. Passiv versteckt den Handelnden – schlecht für eine Security-Marke.
- **Kein vages „Klicken Sie hier“ / „Weiter“:** sagt nichts über das Ergebnis. Das
  Button-Label benennt die Handlung selbst.
- **Pronomen an die Seite anpassen:** CTA-Anrede folgt dem du/Sie der ganzen Seite
  (Verbpaare unten).
- **Friction-Reducer dazu:** „Keine Kreditkarte nötig“ · „Jederzeit kündbar“ · „In 2 Minuten
  startklar“ · „Kein Verkaufsgespräch“. Je größer die Zusage, desto wichtiger.
- **Ein primärer CTA pro Ansicht.** Ein zweiter, leiserer (Ghost-Button) ist für
  Unentschlossene okay.

## 4. du/Sie-Verbpaare (nie mischen)

Wähl eine Spalte und bleib darin – über Hero, Buttons, Formular und Microcopy hinweg.

| du | Sie |
|---|---|
| Teste | Testen Sie |
| Hol dir | Holen Sie sich |
| Starte | Starten Sie |
| Sichere dir | Sichern Sie sich |
| Erfahre | Erfahren Sie |
| Schütze | Schützen Sie |
| Plane | Planen Sie |
| Mach | Machen Sie |
| Buche | Buchen Sie |
| dein / deine | Ihr / Ihre |
| dich | Sie |
| dir | Ihnen |

**Mischfehler (vermeiden):** Hero „Schützen Sie Ihre Marke“, Button gleich daneben „Hol
dir den Report“ – wie bei manchen lokalisierten DACH-Seiten beobachtet. Eine Anrede, eine
Seite.

## 5. Idiomatische Hero-/Headline-Muster

Eine Headline hat **eine** Aufgabe: die nächste Zeile verdienen. Nutzen zuerst, Mechanismus
gleich dahinter, ein Versprechen pro Zeile, konkret statt floskelhaft. Deutsche
Zurückhaltung schlägt englischen Hype – gestapelte Superlative wirken leer; konkrete Belege
überzeugen.

- **Nutzen-first, deklarativ (Segment + Nomen, kein Hype):**
  „Markenschutz für Sicherheitsteams, die genug von Whack-a-Mole haben.“
- **„Mit [Produkt] + Verb“:**
  „Mit [Produkt] verschwinden Fake-Shops, bevor Ihre Kunden darauf hereinfallen.“
- **Imperativ-Versprechen:**
  „Schützen Sie Ihre Marke vor Betrug im Netz.“
- **Outcome + Mechanismus:**
  „Gefälschte Websites erkennen, bevor sie Schaden anrichten – und vom Netz nehmen.“
- **Konkrete Zahl / Beweis-first (nur wenn echt):**
  „1.200 Marken. Keine geklonte Phishing-Seite, die lange online bleibt.“
- **Kontrast / Vorher-Nachher:**
  „Weniger Zeit fürs Jagen von Fakes. Mehr Zeit für echte Bedrohungen.“

**Subhead-Regeln:** erweitert das **eine** Versprechen der Headline, liefert Mechanismus
oder Beleg und leitet zum CTA über. Ein bis zwei Sätze, keine Wiederholung der Headline.

**Wettbewerber-Test:** Könnte ein Mitbewerber den Satz 1:1 übernehmen, ohne dass er falsch
wird? Wenn ja, ist es eine Floskel → ersetzen durch eine Zahl, einen Mechanismus, einen
Namen.

## 6. Harte Regeln vs. umstrittener Stil

**Harte Regeln** (objektive Fehler – immer korrigieren; DIN 5008 / Duden):

- **Anführungszeichen** deutsch `„…“` (99 unten / 66 oben) oder einwärts `»…«` – **nie**
  gerade `"…"` oder englische `"…"`.
- **Kein „in“ vor blanker Jahreszahl:** „in 2024“ ✗ → „2024“ / „im Jahr 2024“ ✓.
- **Kein Deppenleerzeichen:** „Social Media Plattform“ ✗ → „Social-Media-Plattform“ ✓;
  „Marketing Strategie“ ✗ → „Marketing-Strategie“ ✓. Mehrwortbegriffe durchkoppeln.
- **Währung nach dem Betrag mit Leerzeichen:** „1.999,00 €“ ✓ – nicht „€1,999.00“, nicht
  „1,99€“. Dezimalkomma, Tausenderpunkt. **Prozent mit Leerzeichen:** „20 %“, nicht „20%“.
- **Gedankenstrich** `–` (Halbgeviert) mit Leerzeichen; den englischen Geviertstrich `—`
  im Deutschen meiden.

**Umstrittener Stil** (Smells – abwerten, nie verbieten; die Nutzerin darf sie wollen):

- **„Sinn machen“:** Duden markiert es umgangssprachlich, **Eisenberg verteidigt es** als
  wohlgeformtes Deutsch. In polierter Copy lieber „ergibt Sinn“ – aber kein Fehler.
- **Calques** wie „am Ende des Tages“ (at the end of the day), „nicht wirklich“ (not
  really), „einmal mehr“ (once more): wirken übersetzt; biete eine knappere native Wendung,
  aber das ist Stil, kein Fehler.
- **„realisieren“ (= erkennen):** eigentlich „verwirklichen“; die Bundeskanzlei führt es als
  Bedeutungslehnwort. In CTAs/Headlines selten, aber zu beachten.
- **Floskeln** (innovativ, ganzheitlich, maßgeschneidert, „führender/verlässlicher
  Partner“): markieren und durch einen konkreten Fakt ersetzen – ein Qualitäts-Hinweis,
  keine harte Regel.

Wenn du etwas Umstrittenes änderst, nenn den Grund in einem Halbsatz – keine Belehrung.

## 7. Schnell-Check

- [ ] Nutzen zuerst; ein Versprechen pro Headline.
- [ ] CTA = Aktionsverb + Nutzen; ein primärer CTA; Friction-Reducer dabei.
- [ ] CTA native (über die Map abgebildet), keine Übersetzung von „Get started“/„Learn more“.
- [ ] du/Sie laut Markenbrief, über alle Optionen hinweg konsistent – kein Mischen.
- [ ] Wettbewerber-Test bestanden (kein Satz, den ein Mitbewerber 1:1 übernehmen könnte).
- [ ] Typografie korrekt: `„…“`, „20 %“, „1.999,00 €“, kein „in 2024“, kein
      Deppenleerzeichen.
- [ ] Smells abgewertet (mit Grund), nicht verboten.
- [ ] Deutsche Zurückhaltung statt gestapelter Superlative.

> Für tiefere Regeln zu du/Sie, Typografie und Floskeln: die Skills `du-sie-check`,
> `typografie-de` und `texten-de`. Diese Datei ist eigenständig – sie braucht sie nicht.
