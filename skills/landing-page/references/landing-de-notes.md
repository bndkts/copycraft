# Deutsche Landingpage – Umsetzungsregeln (native, nicht übersetzt)

Lies diese Datei, **bevor** du eine deutsche Landingpage schreibst. Sie ist die
Ausführungsebene fürs Deutsche; die sprachneutrale Struktur steht in `page-structure.md`,
der Ablauf in `SKILL.md`. Kernhaltung: Eine deutsche Seite wird **native gedacht**, nicht
aus dem Englischen übersetzt – von der Headline bis zum letzten Button. Schreib aus der
**Bedeutung**, nicht Satzbaustein für Satzbaustein.

Du bist eine erfahrene deutsche Werbetexterin. Du schreibst klar, idiomatisch, ohne
Denglisch und ohne Floskeln. Du orientierst dich am Markenbrief (`brand-brief.md`), wenn er
existiert – er gewinnt immer gegen jede Default-Regel hier.

Diese Datei ist **selbsttragend**: Alles, was du für eine deutsche Seite brauchst – Anrede,
CTAs, Typografie, Floskeln, Verbalstil – steht hier. Alle Beispiele sind **selbst
konstruiert** (Domäne: Markenschutz / Brand Protection, B2B-Security), keine fremde Copy.
Vor Veröffentlichung von einer muttersprachlichen Texterin prüfen lassen.

## 1. du oder Sie

Die **Todsünde** auf einer deutschen Seite ist das **Mischen** von du und Sie innerhalb der
Journey: Der Hero siezt, das Formular oder ein späterer Abschnitt duzt. Bei einer Security-
oder Compliance-Marke wirkt das unsauber und schadet dem Vertrauen sofort. Genau das passiert
leicht auf einer Landingpage, weil Hero, Features, FAQ und Final-CTA oft nacheinander
entstehen. **Leg die Anrede einmal oben fest** und prüf jede Sektion, jeden Button, jede
Microcopy gegen die Verbpaar-Tabelle.

Quelle der Wahrheit ist der **Markenbrief**. Steht dort nichts und sagt die Nutzerin nichts,
**einmal nachfragen** – dann konsequent bleiben. Ohne Vorgabe und bei formellem B2B-Publikum:
**Sie** als Default. Ein falsches Sie kostet weniger als ein falsches du (Strzelecki,
CleverReach: „Ein 'Sie' tut niemandem weh“; Appinio 2019, n ≈ 4.877: 61 % sehen Siezen als
Respekt). **du** nur, wenn die Marke bewusst Gründer/SMB-Self-Serve adressiert und es
**überall** durchzieht (Muster: sevDesk, Trade Republic, N26).

**Anrede ≠ Tonalität:** Man kann warm siezen und steif duzen. Nähe entsteht aus den
Formulierungen, nicht aus dem Pronomen. Ein modernes, warmes Sie ist kein Amtsdeutsch.

**du/Sie-Verbpaare (nie mischen – eine Spalte wählen, über die ganze Seite halten):**

| du | Sie |
|---|---|
| Teste | Testen Sie |
| Hol dir | Holen Sie sich |
| Starte | Starten Sie |
| Sichere dir | Sichern Sie sich |
| Erfahre | Erfahren Sie |
| Schütze | Schützen Sie |
| Plane / Buche | Planen Sie / Buchen Sie |
| Mach | Machen Sie |
| Leg los | Legen Sie los |
| Entdecke | Entdecken Sie |
| dein / deine | Ihr / Ihre |
| dich | Sie |
| dir | Ihnen |

**Mischfehler (konstruiert, zu vermeiden):**
> Hero: „Schützen Sie Ihre Marke vor Betrug im Netz.“ (Sie)
> Final-CTA weiter unten: „Sichere dir jetzt deinen Report.“ (du)

Eine Seite, eine Anrede. Für die systematische Prüfung zusammengesetzter Texte gibt es den
Skill `du-sie-check` – diese Datei kommt aber ohne ihn aus.

## 2. Native CTAs

Englische CTAs werden **nicht übersetzt**, sondern auf die Wendung **abgebildet**, die ein
deutsches Produkt tatsächlich nutzt. „Get started“ wird zu „Jetzt starten“, nicht zu
„Bekomme gestartet“.

| Englischer Ursprung | Natives Deutsch |
|---|---|
| Get started / now | Jetzt starten · Jetzt loslegen · Los geht's |
| Learn more | Mehr erfahren · Mehr dazu |
| Sign up (free) | Kostenlos registrieren · Jetzt kostenlos anmelden |
| Start free trial / Try for free | Kostenlos testen · 30 Tage kostenlos testen · Kostenlos ausprobieren |
| Book / Request a demo | Demo vereinbaren · Demo anfragen · Live-Demo ansehen |
| Contact sales | Vertrieb kontaktieren · Kontakt aufnehmen |
| Buy now | Jetzt kaufen |
| Download | Jetzt herunterladen · Hol dir die App / Holen Sie sich die App |
| Get a quote | Angebot anfordern |

**Auf der Landingpage zeigt jeder CTA auf dieselbe eine Aktion** (z. B. „Demo vereinbaren“).
Statt eines generischen lohnt fast immer der themenspezifische CTA – er nennt den Nutzen
statt nur die Handlung. Security-Beispiele (konstruiert): „Markenschutz starten“ ·
„Marke jetzt prüfen lassen“ · „Bedrohungen prüfen lassen“ · „Fake-Shops melden“.

**CTA-Handwerk:**
- **Aktionsverb zuerst:** Hol dir / Starte / Teste / Sichere dir / Buche (oder gesiezt:
  Holen Sie sich, Starten Sie, Testen Sie …).
- **Nutzen anhängen:** „Jetzt starten & 20 % sparen“ schlägt das bloße „Jetzt starten“.
- **Kein schwaches Passiv:** „Hier können Informationen erhalten werden“ → „Hol dir jetzt
  alle Infos.“
- **Kein vages „Klicken Sie hier“ / „Weiter“:** sagt nichts über das Ergebnis.
- **Friction-Reducer am finalen CTA:** „Keine Kreditkarte nötig“ · „Jederzeit kündbar“ ·
  „In 2 Minuten startklar“ · „Kein Verkaufsgespräch“. Je größer die Zusage, desto wichtiger.
- **Ein primärer CTA pro Ansicht.** Ein zweiter, leiserer (Ghost-Button) ist okay; ein
  zweiter gleichwertiger ist ein Fehler.

## 3. Floskeln und Verbalstil

**Deutsche Zurückhaltung schlägt englischen Hype.** Konkrete Belege und Substanz überzeugen
im deutschen B2B; gestapelte Superlative wirken leer und unseriös. Seriosität kommt aus
fehlerfreien Texten, belegten Fakten und echten Kundenzitaten – nicht aus langen Wörtern.

**Floskeln-Test:** Könnte ein Mitbewerber den Satz 1:1 übernehmen, ohne dass er falsch wird?
→ ja = Floskel = streichen und durch eine Zahl, einen Mechanismus, einen Namen ersetzen.

**Floskeln-Blacklist (jede durch konkreten Fakt ersetzen):**
- **Adjektive:** innovativ, ganzheitlich, maßgeschneidert, kundenorientiert, einzigartig,
  nachhaltig, hochwertig, zukunftssicher, erstklassig, führend, leistungsstark,
  revolutionär, disruptiv, skalierbar (unbelegt).
- **Phrasen:** „maßgeschneiderte Lösungen“, „Lösungen aus einer Hand“, „Wir sind für Sie
  da“, „Ihre Zufriedenheit ist unser oberstes Ziel“, „Der Kunde steht im Mittelpunkt“,
  „Qualität, die überzeugt“, „Wir setzen Maßstäbe“, „Erleben Sie den Unterschied“,
  „Wir gehen die Extrameile“, „Ihr starker/verlässlicher/kompetenter Partner“.
- **Buzzword-Bingo (oft Denglisch):** Synergien, Best Practices, Pain Points, Touchpoints,
  Customer Journey, Deep Dive, Game-Changer, proaktiv, Mindset, „out of the box“,
  „next Level“.

**Verbalstil statt Nominalstil.** Substantivierungen (auf -ung, -heit, -keit, -tion, -tät)
machen Copy steif und distanziert – auf einer Landingpage Gift. Zurück zu Verben:

| Nominalstil (steif) | Verbalstil (lebendig) |
|---|---|
| Zur Durchführung der Analyse | Um zu analysieren |
| Die Implementierung der Optimierung | Wir optimieren |
| Unter Berücksichtigung der Ergebnisse | Wir berücksichtigen die Ergebnisse |
| Die Erkennung gefälschter Seiten erfolgt automatisch | Wir erkennen gefälschte Seiten automatisch |

**Funktionsverbgefüge → starke Verben:** Beachtung schenken → beachten · Hilfe leisten →
helfen · in Anspruch nehmen → beanspruchen · zur Sprache bringen → ansprechen · eine
Entscheidung treffen → entscheiden · zur Durchführung → durchführen.

**Aktiv vor Passiv** (Faustregel ≤ ~10 % Passiv). „Die Fake-Seite wird erkannt“ versteckt
den Handelnden – schlecht für eine Security-Marke, deren ganzer Pitch *„wir tun das für
Sie“* ist. Besser: „Wir erkennen die Fake-Seite.“

**Kurze, abwechslungsreiche Sätze.** Schnitt deutlich unter ~15 Wörtern, harte Obergrenze
~25. Keine Schachtelsätze (max. ein Komma/eine Nebensatz-Ebene). Web: max. 3–4 Sätze pro
Absatz, eine Idee pro Absatz, Kernaussage nach vorn (invertierte Pyramide) – ~80 % der Leser
scannen nur.

**Vorher → Nachher (konstruiert):**
> ✗ „Mit unserer innovativen, ganzheitlichen Lösung schützen wir Ihre Marke umfassend in der
>   heutigen digitalen Welt.“ (Floskeln + Nominalstil + Hype)
> ✓ „Wir finden gefälschte Shops und Phishing-Seiten, die Ihre Marke missbrauchen – und
>   nehmen sie vom Netz.“ (konkret, verbal, ein Versprechen)

## 4. Typografie-Kurzregeln

Diese Regeln sind **harte Regeln** (objektive Korrektheit nach DIN 5008 / Duden), außer wo
als Variante markiert. Auf einer öffentlichen Seite fallen Verstöße sofort auf und wirken
unseriös.

- **Anführungszeichen:** deutsch `„…“` (99 unten / 66 oben) **oder** einwärts gerichtete
  Guillemets `»…«`; verschachtelt `‚…'`. **Nie** gerade `"…"` oder englische `"…"`.
- **Striche:** Gedankenstrich `–` (Halbgeviert) **mit Leerzeichen**; Bindestrich `-` für
  Komposita; Bis-Strich `–` ohne Leerzeichen für Bereiche (10–18 Uhr, 2024–2026). Den
  englischen Geviertstrich `—` im Deutschen meiden.
- **Komposita mit Englisch:** englische Substantive großschreiben (das Feature, das
  Dashboard, das Monitoring). **Kein Deppenleerzeichen:** „Social Media Plattform“ ✗ →
  „Social-Media-Plattform“ ✓; „Marketing Strategie“ ✗ → „Marketing-Strategie“ ✓.
  Mehrwortbegriffe durchkoppeln: Work-Life-Balance. E-Mail, E-Book, E-Commerce.
- **Zahlen/Währung:** Dezimalkomma, Tausenderpunkt → 1.000,00 € / 1.250.000,99 €.
  Währungssymbol **nach** dem Betrag mit Leerzeichen: „1.999,00 €“ ✓ – nicht „€1,999.00“,
  nicht „1,99€“. Prozent mit Leerzeichen: „20 %“, nicht „20%“.
- **Datum/Uhrzeit:** DD.MM.YYYY (29.06.2026) oder „29. Juni 2026“; nie US-Format. 24 h,
  „14:30 Uhr“. **Nie „in 2024“** → „2024“ / „im Jahr 2024“.

## 5. Harte Regeln vs. umstrittener Stil

**Harte Regeln** (Fehler – immer korrigieren): die Typografie aus §4; kein gemischtes
du/Sie; keine erfundenen Belege (echte Zahl oder Platzhalter `[ ]`); ein primärer CTA pro
Ansicht; nicht gegen den Markenbrief verstoßen.

**Umstrittener Stil** (Smells – abwerten, **nie verbieten**; die Nutzerin darf sie wollen,
und Fachleute sind uneins). Bei einer Änderung den Grund in einem Halbsatz nennen, keine
Belehrung. Quellen jeweils genannt:

- **„Sinn machen“:** Duden markiert es umgangssprachlich, **Eisenberg verteidigt es** als
  wohlgeformtes Deutsch. In polierter Copy lieber „ergibt Sinn“ – aber kein Fehler.
- **Calques:** „am Ende des Tages“ (at the end of the day) → „letztlich/unterm Strich“;
  „nicht wirklich“ (not really) → „eigentlich nicht/kaum“; „einmal mehr“ (once more) →
  „erneut“. Wirken übersetzt; eine knappere native Wendung ist besser, aber es ist Stil.
- **„realisieren“ (= erkennen):** heißt eigentlich „verwirklichen“; die **Bundeskanzlei**
  führt es als Bedeutungslehnwort. Besser „erkennen/bemerken“.
- **„die Extrameile gehen“ / „ein Problem adressieren“:** Calque bzw.
  Bedeutungsverschiebung; „sich besonders einsetzen“ / „ein Problem angehen“.
- **Scheinanglizismen** als deutsche Wörter behandeln, nicht „korrigieren“: Handy,
  Homeoffice, Shitstorm, Homepage (Deutsche meinen oft die ganze Website).
- **Floskeln** (siehe §3): markieren und durch einen Fakt ersetzen – Qualitäts-Hinweis,
  keine harte Regel.

## 6. Schnell-Check

Pro Sektion und für die ganze Seite:

- [ ] Anrede einmal festgelegt (Markenbrief), über **alle** Sektionen konsistent – kein
      Mischen von du/Sie.
- [ ] Jeder CTA native (über die Map abgebildet), nicht aus „Get started“/„Learn more“
      übersetzt; alle CTAs zeigen auf **dieselbe** eine Aktion.
- [ ] Aktionsverb zuerst, Nutzen angehängt; Friction-Reducer am finalen CTA.
- [ ] Floskeln-Test bestanden – kein Satz, den ein Mitbewerber 1:1 übernehmen könnte.
- [ ] Verbalstil statt Nominalstil; Aktiv vor Passiv; kurze Sätze; eine Idee pro Absatz,
      Kernaussage vorn.
- [ ] Belege echt oder als `[ ]`-Platzhalter markiert – nichts erfunden.
- [ ] Typografie korrekt: `„…“`, „20 %“, „1.999,00 €“, „29.06.2026“, kein „in 2024“, kein
      Deppenleerzeichen, kein Geviertstrich `—`.
- [ ] Deutsche Zurückhaltung statt gestapelter Superlative; Vertrauen über prüfbare Fakten
      und echte Kundenzitate.
- [ ] Smells (Sinn machen, Calques) abgewertet mit Grund, nicht verboten.

> Tiefere Regeln in den Skills `texten-de`, `typografie-de`, `du-sie-check` und
> `headlines-and-ctas`. Diese Datei ist eigenständig – sie braucht sie nicht.
