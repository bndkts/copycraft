# Native CTA-Map und du/Sie-Verbpaare

Diese Datei hält lokalisierte Handlungsaufforderungen (CTAs) **nativ** und die Anrede
**konsistent**. Lies sie, sobald Buttons, Links oder Aufforderungen im Text vorkommen.

Standard-CTAs sind die häufigste Quelle für Translationese: „Get started“, „Learn more“,
„Sign up“ werden reflexhaft wörtlich übersetzt und klingen sofort übersetzt. Übertrage
stattdessen die **Funktion** des CTA in eine deutsche Formel.

## 2. CTA-Handwerk

- **Aktionsverb zuerst:** Hol dir / Holen Sie sich, Starte / Starten Sie, Teste / Testen
  Sie, Sichere dir / Sichern Sie sich, Buche / Buchen Sie.
- **Nutzen anhängen:** „Jetzt starten & 20 % sparen“ wirkt stärker als bloßes „Jetzt
  starten“. Der CTA verspricht das Ergebnis, nicht nur die Handlung.
- **Risiko reduzieren:** direkt am CTA den wahrgenommenen Preis des Ja senken – „ohne
  Kreditkarte“, „jederzeit kündbar“, „30 Tage kostenlos“, „in 2 Minuten startklar“.
- **Kein schwaches Passiv:** „Hier können Informationen erhalten werden“ ✗ → „Hol dir
  jetzt alle Infos“ ✓.
- **Nichts Vages:** „Klicken Sie hier“, „Weiter“, „Mehr“ sagen nichts über das Ergebnis.
- **Pronomen zur Seite passend:** Das CTA-Pronomen folgt der Anrede der ganzen Seite
  (du **oder** Sie), nie gemischt – siehe unten.

---

## 3. du/Sie – die Anrede konsequent halten

Die Anrede ist **keine** Tonalität: Man kann warm siezen und steif duzen. Nähe entsteht aus
den Formulierungen, nicht aus dem Pronomen. Die **Todsünde** ist das **Mischen** von du und
Sie innerhalb einer Journey – bei Vertrauensmarken (Security, Finanzen) direkt schädlich.

**Entscheidung:** Sie steht im `brand-brief.md`. Fehlt der Brief, frag einmal nach oder
triff eine begründete Annahme und mach sie transparent. Faustregel: Ein falsches Sie kostet
weniger als ein falsches du („Ein ‚Sie’ tut niemandem weh“). Für tiefergehende Audits ist
die Skill `du-sie-check` zuständig; für die Lokalisierung reicht: **eine** Anrede wählen
und sie überall durchziehen.

### Verbpaare (links du / rechts Sie – nie mischen)

| du | Sie |
|---|---|
| Teste | Testen Sie |
| Hol dir | Holen Sie sich |
| Starte | Starten Sie |
| Sichere dir | Sichern Sie sich |
| Erfahre | Erfahren Sie |
| Schütze | Schützen Sie |
| Plane | Planen Sie |
| Buche | Buchen Sie |
| Mach | Machen Sie |
| dein / deine | Ihr / Ihre |
| dich | Sie |
| dir | Ihnen |

### Mischfehler (zu vermeiden)

Hero siezt – „Schützen Sie Ihre Marke“ – und das Formular darunter duzt plötzlich –
„Erfahre, wie du Zeit sparst“. Das ist der klassische, vertrauensschädliche Bruch. Prüfe
am Ende jede Aufforderung, jedes Possessivpronomen und jede Anrede auf **eine** Linie.

> Hinweis: Die du/Sie-Daten und Markenbeispiele sind richtungsweisend (Studien 2019–2022),
> nicht tagesaktuell. Im Zweifel entscheidet die Zielgruppe – und immer der Markenbrief.
